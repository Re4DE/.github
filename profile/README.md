# Re4DE

This GitHub organization provides the code base and documentation of the Reference System for Data Exchange in the Energy Sector (Re4DE).

## Licensing

Each repository contains its own license file.  
Please refer to the respective licensing information in the individual projects.

## Contributing

Re4DE is a research-driven project. The repositories are publicly available, but this is not an actively managed open-source community. Nevertheless, feedback, questions, and selected contributions are welcome.

For code and repository questions, please contact the maintainers or open an issue in the relevant repository:
- [@kosrenDQ](https://github.com/kosrenDQ)
- [@darioklapp](https://github.com/darioklapp)

For general questions about the Re4DE project or the potential of dataspaces in the German energy sector, please reach out to the [Future Energy Lab](https://www.future-energy-lab.de/).

For contribution guidelines, please refer to the [CONTRIBUTING.md](https://github.com/Re4DE/.github/blob/main/CONTRIBUTING.md) file.
